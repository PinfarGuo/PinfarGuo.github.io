Using Covid Data obtained from 'Our World in Data' I created several SQL queries to see the correlation between Infections and Deaths, and displayed it on a simple Tableau Dashboard.

  • SQL - created 4 simple queries displaying Covid info in regards to infections and deaths. Also, I created additional queries to further display SQL skills.
  • Tableau - created an simple dashboard displaying skills using text tables, stacked bars, maps, and line charts with forcasting.

SQL and Tableau

First: a query to display some basic global numbers:

Select SUM(new_cases) as TotalCases,
	SUM(CAST(new_deaths as int)) as TotalDeaths,
	SUM(CAST(new_deaths as int))/SUM(new_cases)*100 as DeathPercentage
From CovidPortfolioProject..CovidDeaths
Where continent IS NOT NULL
Order By 1,2
;

Second: query for the count of death count per continent

Select continent,
	SUM (CAST (new_deaths as int)) as TotalDeathCount
From CovidPortfolioProject..CovidDeaths
Where continent IS NOT NULL
Group By continent
Order By TotalDeathCount desc
;

Third: query containing location information in regards to infection count to be used on a map

Select location, population, 
	MAX(total_cases) as HighestInfectionCount,
	MAX(total_cases/population)*100 as PercentageInfected
from CovidPortfolioProject..CovidDeaths
Group By location, population
order By PercentageInfected desc
;

Next: query containing location information and date information regarding infection so that we can see the progression in a line chart

Select location, population, date,
	MAX(total_cases) as HighestInfectionCount,
	MAX(total_cases/population)*100 as PercentageInfected
From CovidPortfolioProject..CovidDeaths
Group By location, population, date
order By PercentageInfected desc
;
